Este proyecto fue desarrollado en Java aplicando múltiples interfaces para representar distintos roles dentro de una empresa.

El sistema maneja tres tipos de usuarios:

Cajero
Administrador
Supervisor

Para ello se implementaron las interfaces:

Autenticable
Reportable
Gestionable

Cada rol posee diferentes permisos y funcionalidades dependiendo de las interfaces que implementa.
El sistema permite iniciar sesión mediante usuario y contraseña, generar reportes y gestionar datos según el tipo de usuario seleccionado.

También se implementó un menú sencillo en consola para interactuar con el programa.
